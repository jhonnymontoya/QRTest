QRTest is a SwiftUI application that generates varius types QR Codes

## Types of QR

In this example only generates QR Code model 1 and 2 for:

- `Website`
- `SMS`
- `vCard`
- `Social Media`
- `WhatsApp`
- `Calendar`
- `Geolocation`
- `One Time Password (OTP)`
- `Phone number`
- `WiFi`
